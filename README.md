#### idea 插件简介

<ul>定制化java代码开发插件,用于学习插件开发,欢迎交流想法
    <li>根据建表SQL一键生成mapper、model、VO、service、controller代码。支持基础增删查改、列表查询、批量新增</li>
    <ul>
        <li>支持mysql、oracle，需带上COMMENT</li>
        <li>支持生成ibatis版代码</li>
        <li>支持自定义模板</li>
    </ul>
    <li>一键生成类转换的构造函数</li>
    <li>一键翻译（需联网）</li>
</ul>
